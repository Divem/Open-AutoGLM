# Change: Update Web Interface Branding to Terminal Agent

## Why
为了统一Web界面的品牌标识，提供更专业的用户体验，需要将Web页面中的品牌显示从"Phone Agent"更新为"Terminal Agent"。

## What Changes
- 更新HTML页面标题标签（title标签）为"Terminal Agent"
- 更新Web界面导航栏品牌标识为"Terminal Agent"
- 更新欢迎信息显示为"欢迎使用 Terminal Agent"
- 更新聊天界面头部标题为"Terminal Assistant"
- 保持现有的功能和用户体验不变

## Impact
- Affected specs: web-interface, user-branding
- Affected code: web/templates/index.html, web/templates/config.html
- **NON-BREAKING**: 仅更新界面文本，不影响功能逻辑
